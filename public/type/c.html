Sleep Type C

🌑 朝つらい型 のあなたへ

時間は寝ているのに疲れが抜けない。就寝時刻のばらつきや、眠りの浅さ、首まわりの合わない寝具が疑われるタイプです。

今夜からできる、お金のかからないこと

今夜からできるのは「起きる時刻を固定する」こと。寝る時刻より起きる時刻を揃える方が、体内時計は整いやすいです。
