## B. GA4 発行〜差し替え手順

1. GA4管理画面で新規プロパティ「Somni」を作成、データストリームで `somni.sanji-104vt.workers.dev` を登録
2. 発行された測定ID(`G-XXXXXXXXXX`)を控える
3. `public/index.html` の `<head>` 内、コメントアウトされているGA4スクリプト2箇所のIDを差し替え、コメントアウトを解除
4. Search Console 側でもプロパティ追加(URLプレフィックス方式)→ 検証タグは共有のもの(`UucVcbwbG6YhXKLVS3GGS8nVk_egyJCLywDHkw6J-5Q`)が既にHTML内にあるため、追加せず「確認」を押すだけで認証されるはず
5. サイトマップ送信: 単一HTMLサイトのためsitemap.xmlは以下の最小構成で足りる(記事ページを増やしたら都度追記)

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url><loc>https://somni.sanji-104vt.workers.dev/</loc></url>
</urlset>
```

これを `public/sitemap.xml` に配置し、GSCの「サイトマップ」メニューから `sitemap.xml` を送信する。
